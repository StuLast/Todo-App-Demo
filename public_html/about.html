About

The Todo app is a simple javascript based application that creates and tracks todos. The data for the todos is stored locally on your browser. The app is intended only as a demo application.

Privacy

The app does not collect any private information nor store any information on remote servers. All data entered is contained in the browsers local data storage. Users can manually delete all local data storage by following the instructions for their own broswer.

Cookies

This app doesn't use any cookies but does use local storages (see "privacy" above). There is no tracking of usage, no monitoring or sharing of data and no login/session storage.
